# basic-app-nextjs-app-router

## 실행

### 프로젝트 생성

```
npx create-next-app basic-app-nextjs-app-router
//npx create-next-app basic-app-nextjs-app-router --typescript

```

```
cd basic-app-nextjs-app-router
```
